# Analyse BBOB

Projet d'analyse de `PSO`, `GA` et `RANDOMSEARCH-5` sur des fonctions `BBOB` en dimension 20.

## Fichiers importants

- `Analyse-data.ipynb` : notebook principal
- `data_archive_pso/` : donnees PSO
- `data_archive-GA/` : donnees GA
- `data_archive_RANDOMSEARCH-5/` : donnees Random Search
- `figures_bbob/` : figures generees

## Installation

Installer les dependances :

```bash
pip install -r requirements.txt
```

## Utilisation

Lancer Jupyter puis ouvrir le notebook :

```bash
jupyter notebook Analyse-data.ipynb
```

Le notebook contient :

- le chargement des fichiers `.dat`
- l'analyse des runs
- les courbes de convergence
- les comparaisons entre algorithmes
